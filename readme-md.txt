# ランダムグループ振り分けアプリ

シンプルなHTMLとJavaScriptで作られた、メンバーをランダムにグループ分けするためのウェブアプリケーションです。

## 特徴

- メンバー名の入力（改行、カンマ、日本語カンマ区切り）
- グループ数の指定
- ランダムな振り分け
- レスポンシブデザイン
- 外部依存ライブラリは最小限（TailwindCSSのみ、CDN経由）

## 使い方

1. メンバー名をテキストエリアに入力します（各名前は改行、カンマ「,」または日本語カンマ「、」で区切る）
2. 作成したいグループ数を指定します
3. 「振り分け開始」ボタンをクリックします
4. 振り分け結果が表示されます

## デモ

GitHub Pagesでホスティングすることで、すぐに使えるデモを提供できます。

## インストール方法

このリポジトリをクローンするか、ダウンロードするだけですぐに使えます：

```bash
git clone https://github.com/yourusername/random-group-assignment.git
cd random-group-assignment
```

## 開発方法

1. このリポジトリをクローンします
2. `index.html`をブラウザで開くか、シンプルなWebサーバーでホスティングします

```bash
# Pythonを使った簡易サーバーの例
python -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` にアクセスします。

## デプロイ方法

### GitHub Pagesを使う場合

1. GitHubでリポジトリを作成します
2. このプロジェクトのファイルをアップロードします
3. リポジトリの「Settings」タブから「Pages」を選択します
4. ソースとして「main」ブランチを選択し、保存します

### Netlifyを使う場合

1. Netlifyにアカウント登録してログインします
2. 「New site from Git」を選択します
3. GitHubを選択し、作成したリポジトリを選択します
4. デプロイ設定を確認し、「Deploy site」をクリックします

## カスタマイズ

- `index.html`のスタイルタグでデザインをカスタマイズできます
- `script.js`を編集して、振り分けロジックを変更できます
